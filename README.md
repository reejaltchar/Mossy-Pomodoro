# Mossy Pomodoro — Focus Timer

A cozy forest-themed Pomodoro timer designed to help you stay focused while maintaining a calm and relaxing workspace.

## 🌐 Live Demo

🔗 https://mossy-pomodoro.netlify.app/

## Features

- ⏱️ Pomodoro, Short Break, and Long Break modes
- 🌿 Cottagecore / forest-inspired design
- 🌙 Dark and Light mode toggle
- ✅ Task management system
- 🎯 Focus task selection
- 📊 Daily session tracking
- 🎵 Embedded Spotify player
- 🔔 Browser notifications and timer alerts
